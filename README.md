# Visualización de Datos de Vehículos Usados

Esta es una aplicación web construida con Streamlit que permite visualizar y explorar datos sobre vehículos usados.

## Funcionalidades

- Visualización de un histograma de los años de fabricación `model_year`.
- Visualización de un diagrama de dispersión de `odometro` vs `precio`.
- Visualización extra: gráfico de cajas (box plot) del `precio` por tipo de `vehiculo`.
- Permite seleccionar qué visualización deseas ver usando casillas de verificación.

Esta herramienta es útil para tener una comprensión visual de las tendencias y distribución de datos de vehículos usados en el dataset proporcionado.
